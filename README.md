# Bostarek Phonics

Launch page for Bostarek Phonics, an educational initiative for young learners
with English as a foreign language.

The site is built as a static GitHub Pages page. The launch page currently
includes:

- Milky Galaxy visual background
- Dyslexia-friendly font choices
- Orange and blue color palette
- Contact email: <bostarek@gmail.com>
- "Get notified" mailto button
